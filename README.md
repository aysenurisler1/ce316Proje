# Integrated Assignment Environment (IAE)

Integrated Assignment Environment (IAE) is a JavaFX-based desktop application developed for managing and evaluating programming assignments.

## Project Purpose

The main purpose of this project is to automate the assignment evaluation workflow by allowing instructors to:

- Create and manage projects
- Configure compilation and execution settings
- Upload and process student submissions
- Run and evaluate assignments automatically
- Compare outputs with expected results
- Export results into CSV format

## Technologies Used

- Java 21
- JavaFX
- Maven
- SQLite
- IntelliJ IDEA

## Features

### Project Management
- Create and manage programming assignments
- Store project configurations
- Organize submission folders

### Configuration System
- Custom compile commands
- Custom run commands
- Language-specific configurations

### Submission Processing
- ZIP extraction support
- Automatic source file detection
- Assignment execution workflow

### Result Management
- Store execution results
- Status tracking
- CSV export support

### User Interface
- JavaFX desktop interface
- Multiple management screens
- Help/About section
- Modern CSS styling
